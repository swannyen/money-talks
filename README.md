# Money Talks
updated: 19 Nov 2025

Money Talks is a repo containing codes I need to help me have a good overview and monitoring of the investments I have.

## How to run the Dashboard

1. Install all dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit app:
```bash
streamlit run app.py
```
This will open the dashboard in your local web browser.
